# CortexNote - NotebookLM Clone

CortexNote — это мощное локальное веб-приложение в стиле **Google NotebookLM** и **OpenAI Prism**, построенное на Next.js. Оно позволяет анализировать документы, генерировать конспекты, аудио-подкасты и презентации с помощью ИИ. 

## 🚀 Ключевые возможности

- 📄 **Мультимодальный анализ**: Загрузка и анализ PDF документов, текстовых файлов и изображений.
- 📝 **Автоматическая ИИ-суммаризация**: Умная выжимка смысла с поддержкой редактирования сгенерированного Markdown.
- 🎙️ **Аудио-подкасты**: Генерация аудио-обсуждений (скриптов) на основе загруженных материалов.
- 📊 **Генерация презентаций**: Автоматическое создание структуры слайдов.
- ➗ **Поддержка LaTeX**: Корректное отображение математических формул в конспектах.

### 🌟 Новые функции (Prism Update)
- 🎨 **Minimalist UI**: Полностью переработанный интерфейс в стиле чистого минимализма (Dark Mode, фокус на контенте).
- 💾 **Локальная База Данных**: Встроенная SQLite (`better-sqlite3`) для мгновенного и надежного сохранения всех ваших проектов и исходников. Автосохранение при вводе текста!
- 🔐 **Встроенная система аккаунтов**: Локальная регистрация и авторизация (хэширование `bcryptjs`, сессии JWT `jose`). Никаких зависимостей от сторонних OAuth провайдеров!
- 🌍 **Explore Community**: Единая лента всех проектов, где вы можете просматривать и изучать конспекты других пользователей системы.

## 🛠️ Технологии

- **Framework:** Next.js 16 (App Router)
- **AI:** Google Gemini API (или OpenRouter для `gemini-2.5-flash-lite`)
- **Database:** SQLite (`better-sqlite3`)
- **Auth:** Custom JWT-sessions (`jose`, `bcryptjs`)
- **Styling:** Tailwind CSS (Neutral Dark theme)
- **UI:** React 19, Lucide Icons
- **Markdown:** react-markdown с поддержкой KaTeX

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/YOUR_USERNAME/CBER_cvs.git
cd CBER_cvs/notebook-lm-web
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Настройка переменных окружения

Создайте файл `.env.local` в корне папки `notebook-lm-web` и добавьте:

```env
# Для генерации контента
GOOGLE_API_KEY=ваш_ключ
# ИЛИ, если используете OpenRouter:
OPENROUTER_API_KEY=ваш_openrouter_ключ

# Секрет для шифрования пользовательских сессий
SESSION_SECRET=super_secret_key_for_jwt
```

### 4. Инициализация базы данных

База данных SQLite создастся автоматически при первом запуске приложения и логине, но вы можете убедиться в ее инициализации:
```bash
npm run dev
```
Файл базы данных появится по пути `data/notebooks.db`.

### 5. Запуск сервера разработки

Откройте [http://localhost:3000](http://localhost:3000) в браузере. Вас встретит экран входа. Зарегистрируйте первый аккаунт и начните работу!
